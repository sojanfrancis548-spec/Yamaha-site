## Features

- Hero section with Yamaha R-Series showcase
- 3D flip card model cards — R15, MT-15, R7
- Photo gallery
- Contact / test ride booking form
- Admin panel to view all form submissions
- Session-based login for admin

## Getting Started

### 1. Install dependencies
pip install flask

### 2. Run the app
python app.py

Visit http://127.0.0.1:5000

### 3. Admin login
Go to http://127.0.0.1:5000/admin/login
